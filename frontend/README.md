# Frontend (HTML + CSS + Vanilla JS)

Frontend реализован без JS-фреймворков: **HTML + CSS + Vanilla JS** + **Bootstrap 5**.

## Страницы

- `events.html` — список событий (фильтрация + пагинация)
- `event-details.html` — детали события + программа мероприятия
- `ticket.html` — покупка билета
- `login.html` — вход
- `register.html` — регистрация
- `forgot-password.html` — восстановление пароля
- `profile.html` — профиль пользователя
- `my-tickets.html` — мои билеты
- `admin.html` — админ-панель (CRUD мероприятий, программа, билеты, заказы)

## Пошаговый запуск

1. Перейдите в папку фронтенда:
   ```bash
   cd frontend
   ```
2. Запустите статический сервер:
   ```bash
   python3 -m http.server 8081
   ```
3. Откройте:
   - `http://localhost:8081/events.html`

## Роли и доступ

- Гость: `События`, `Вход`.
- Пользователь после входа: `События`, `Профиль`, `Мои билеты`, `Выход`.
- Админ: те же страницы + `Админ`.
